## 🚀 **Quick Start (5 Minutes)**

### 1. **System Initialization**
```bash

python3 iza start
```

### 2. **Your First Commands**
```bash

# Find problems to solve
python3 iza scan problems --global

# Start daily learning

# Generate revenue report  
python3 iza revenue

```

